# House Price Prediction using Regression Models

## Project Overview
This project predicts house prices using different machine learning regression models. 
The goal is to analyze housing data and build models that can accurately predict house prices based on given features.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Support Vector Regressor (SVR)

## Project Steps
1. Import required libraries
2. Load and explore the dataset
3. Data preprocessing
4. Encode categorical features using OneHotEncoder
5. Split data into training and testing sets
6. Train regression models
7. Evaluate model performance using R² Score

## Evaluation Metrics
- R² Score
- Mean Squared Error (MSE)

## How to Run the Project
1. Clone the repository
2. Install required libraries
3. Open the notebook file and run the cells
